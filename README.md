## Overview

An emoji search using a json file.

## Packages

- React

## Running the project

Clone this repository :

```

git clone https://github.com/bangtam1994/emoji-search.git

cd emoji-search

```

To install packages :

```
npm install
```

or

```
yarn add
```

When installation is complete, run the project with:

```
yarn start or npm start
```

## Deployment
